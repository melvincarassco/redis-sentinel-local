Getting Started with DevBrain CLI

DevBrain comes with a zero-dependency, local python runner. Download the utility devbrain.py and place it in your local code repository root.

1. Check Available Rules

List all remote templates directly from the catalog:

python3 devbrain.py list

2. Sync a Rule

Download and write a rule to `.cursorrules` or `.agents.md` locally:

python3 devbrain.py sync clerk-sso-nextjs

3. Initialize a Custom Workspace Rule

Scaffold a clean template in your repository root:

python3 devbrain.py init

4. Setup Graphify Codebase Indexing

Initialize a tailored `.graphify.json` to optimize context search costs:

python3 devbrain.py graphify
